This are [Pharo](https://github.com/pharo-project/pharo) bindings for [libbson](https://mongoc.org/libbson/current) 
library, generated using [pharo-cig](https://github.com/estebanlm/pharo-cig).  

The generation recipe can be found at the class comment of [LibBson class](https://github.com/estebanlm/pharo-libbson/blob/main/src/LibBSON/LibBson.class.st).
